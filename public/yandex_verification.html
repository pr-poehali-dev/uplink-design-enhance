Файл для подтверждения прав в Яндекс.Вебмастере

Этот файл используется для подтверждения владения сайтом в Яндекс.Вебмастере.

После получения кода верификации от Яндекса, замените содержимое этого файла на предоставленный HTML-код.

Инструкция:

  1. Зайдите в Яндекс.Вебмастер
  2. Добавьте ваш сайт
  3. Выберите способ подтверждения "HTML-файл"
  4. Скачайте файл от Яндекса или скопируйте его содержимое
  5. Замените содержимое этого файла на код от Яндекса
  6. Нажмите "Проверить" в Вебмастере

Важно: Не удаляйте этот файл после подтверждения - он нужен для постоянной связи с Яндексом.
