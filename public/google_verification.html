Файл для подтверждения прав в Google Search Console

Этот файл используется для подтверждения владения сайтом в Google Search Console.

После получения кода верификации от Google, замените содержимое этого файла на предоставленный HTML-код.

Инструкция:

  1. Зайдите в Google Search Console
  2. Добавьте ваш сайт
  3. Выберите способ подтверждения "HTML-файл"
  4. Скачайте файл от Google (название будет googleXXXXXXXX.html)
  5. Переименуйте этот файл в название, которое дал Google
  6. Или замените содержимое этого файла на код от Google
  7. Нажмите "Подтвердить" в Search Console

Важно: Не удаляйте этот файл после подтверждения.
